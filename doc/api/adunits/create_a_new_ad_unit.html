AdUnits API

A single advertisement that may be served.

Create a new ad_unit

POST /ad_units

This method creates a new advertisement and stores it in the database.

Parameters

Name Description
ad_unit ad unit
ad_unit[title] Advertisement title
ad_unit[description] Advertisement description
ad_unit[click_url] Click URL
ad_unit[ad_unit_url] Advetisement URL
ad_unit[active] Active advertisement
ad_unit[user_id] User ID
