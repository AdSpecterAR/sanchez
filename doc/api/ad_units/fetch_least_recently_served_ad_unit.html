Ad_Units API

A single advertisement that may be served.

Fetch least recently served ad unit

GET /ad_units/fetch

Retrieve the least recently served advertisement unit and its metadata.

Parameters

Name Description
ad_unit ad unit
ad_unit[title] Advertisement title
ad_unit[description] Advertisement description
ad_unit[click_url_default] Click URL
ad_unit[ad_unit_url] Advertisement URL
ad_unit[active] Active advertisement
ad_unit[user_id] User ID
ad_unit[ad_format] Advertisement format
ad_unit[aspect_ratio_width] Width of aspect ratio
ad_unit[aspect_ratio_height] Height of aspect ratio
ad_unit[last_served_at] Time at which ad unit was last served
ad_unit[click_url_android] Android mobile deep link
ad_unit[click_url_ios] iOS mobile deep link
ad_unit[call_to_action] Call to action engagement
