Impressions API

Within an app session many advertisements may be served, each advertisement of which may receive multiple impressions. Impressions may be received, shown, and or clicked.

Creates a new impression

PUT /impressions

This method creates a new impression.

Parameters

Name Description
impression Impression
impression[ad_unit_id] Ad unit
impression[developer_app_id] Developer App Id
impression[app_session_id] App Session Id
impression[served] Whether ad has been served
impression[served_at] When ad was served
impression[shown] Whether ad has been shown
impression[show_at] When ad was shown
impression[clicked] Whether ad was clicked
impression[clicked_at] When ad was clicked

Request

Headers

Host: example.org
Content-Type: application/x-www-form-urlencoded
Cookie:

PUT /impressions
