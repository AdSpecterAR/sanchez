Users API

Users may open sessions with many developer apps. They can be created by a POST request.

Create new user

POST /users

This method creates a new user.

Parameters

Name Description
user New user
user[first_name] First name
user[last_name] Last name
user[account_type] Developer or Advertiser
user[email] Email account
user[password] Password

Request

Headers

Host: example.org
Content-Type: application/x-www-form-urlencoded
Cookie:

Route

POST /users

Body

user[first_name]=John&user[last_name]=Wooden&user[account_type]=developer&user[email]=johnwooden%40ucla.edu&user[password]=password1&user[user_id]=1
